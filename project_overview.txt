PROJECT SUMMARY
-------------------------------------------------------------------------------
Project Name: Ad Rank Controller (DashboardDemo)
Purpose: A full-stack web application designed for collecting configuration 
         inputs to help rank products at position #1 on an e-commerce marketplace 
         (like Blinkit). It includes a configuration dashboard PLUS an automated 
         bid optimization system.

================================================================================
                               CORE FEATURES
================================================================================

1. CAMPAIGN CONFIGURATION DASHBOARD (Frontend)
----------------------------------------------
The main dashboard allows users to configure advertising campaigns through 
three sections:

SECTION A - Product Selection:
• Product ID/SKU input
• Product Name
• Category selection
• Brand identification
• Current Rank (starting position)
• Target Rank (goal: typically #1)

SECTION B - Ad Type Selection:
• Multiple ad type support:
  - Product Ads
  - Brand Ads
  - Listing Spotlight
  - Recommendation Ads
• Priority levels: Low, Medium, High

SECTION C - Targeting Rules:
• City and Pincode targeting
• Time Slot scheduling (start/end times)
• Day Type selection (Weekday/Weekend)
• Keyword targeting (with bid booster options)
• Negative keyword exclusion
• Category match toggle

2. CAMPAIGN MANAGEMENT
----------------------
• View all saved campaigns in a table format
• CRUD operations for campaigns via REST API

3. BID OPTIMIZATION SYSTEM (Automation)
---------------------------------------
A sophisticated automated bid optimization system with:

Binary Search Strategy (binary-search.js):
• Per-keyword state tracking
• Decreasing percentage strategy (starts at 5%, decreases by 1% on oscillation)
• Max bid ceiling support per keyword
• Convergence detection when optimal bid found
• Oscillation detection for fine-tuning

Key Configuration (config.js):
  MIN_BID: ₹100
  MAX_BID: ₹50,000
  BID_THRESHOLD: ₹50
  BID_PERCENTAGE: 5% (per iteration)
  RANKING_CHECK_INTERVAL: 100ms
  MAX_ITERATIONS: 25
  AI_TRAINING_THRESHOLD: 20 observations
  TARGET_RANK: 1

Campaign Form Automation (campaign-form-automation.js):
5-step form automation using Playwright:
  Step 1: Ad Format (campaign name, objective, ad asset)
  Step 2: Ad Settings (dates, regions, cities)
  Step 3: Product Details (product selection, filters)
  Step 4: Targeting Options (keywords, bids)
  Step 5: Budget Details (strategy, amounts)
• Screenshot capture at each step
• Per-keyword bid support
• Error handling and debugging

4. AI-POWERED PREDICTION
------------------------
• Gemini AI integration for bid prediction
• Training threshold of 20+ observations
• Learning from historical bid-to-rank data

5. REPORT GENERATION
--------------------
• PDF report generation using PDFKit
• Includes:
  - Campaign details
  - Optimization settings
  - Per-keyword results table
  - Savings summary with percentages
  - Optimization history

================================================================================
                              DATA MODELS
================================================================================

CAMPAIGN SCHEMA (MongoDB)
-------------------------
{
  productId: String (required),
  productName: String (required),
  category: String (required),
  brand: String (required),
  currentRank: Number (min: 1),
  targetRank: Number (default: 1),
  adTypes: [{
    type: ['Product Ads', 'Brand Ads', 'Listing Spotlight', 'Recommendation Ads'],
    priority: ['Low', 'Medium', 'High']
  }],
  targeting: {
    city: String,
    pincode: Number,
    timeSlot: { start: String, end: String },
    dayType: ['Weekday', 'Weekend'],
    keywords: [String],
    categoryMatch: Boolean
  },
  createdAt: Date
}

TYPESCRIPT INTERFACES
---------------------
• Campaign - Full campaign object
• CampaignFormData - Form submission data
• AdType - Ad type with priority
• Keyword - Keyword with bid details
• Targeting - Targeting configuration
• Product - Product information
